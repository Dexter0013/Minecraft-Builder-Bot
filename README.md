# 🤖 Minecraft AI Builder (2026 Edition)

A lightweight, high-performance bridge between **Minecraft Bedrock Edition** and **Google Gemini 2.5 Flash**. This bot listens to your in-game chat and translates natural language requests into instant building commands.

## ✨ Features
- **Instant Building:** "Build a 10x10 oak house" -> Bot executes `/fill` and `/setblock` commands immediately.
- **Safety First:** Built-in filters to prevent TNT griefing or massive world-crashing fills.
- **Cloud Ready:** Optimized for zero-cost hosting on Render/Railway using a native Node.js pipeline.
- **2026 Tech Stack:** Powered by Gemini 2.5 Flash for superior spatial reasoning and building logic.

## 🚀 Quick Start
1. `/connect your-bot-url.com` in Minecraft.
2. Type "Build a stone tower" in chat.
3. Watch the AI work!
